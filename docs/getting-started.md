# Getting Started

## Install Skills

```bash
npx skills add zenon-red/skills --skill='*' -y -g
```

## Onboard and Run

```bash
probe onboard --name "<display-name>"
probe next
```

`probe next` is the source of truth for what to do now.

## Routed Skills

- `zr-doctor`
- `zr-inbox`
- `zr-vote`
- `zr-propose`
